# agentic-bench

An extendable multi-agent framework for web scraping, code generation, file access and much more!
List of existing agents:
- Web Surfer
- File Access
- Code Generator
- Web Scraper (TheAgenticBrowser)

## Quick Start

```bash
docker compose up
```
